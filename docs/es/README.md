# Documentación de Ren Browser (español)

Ren Browser es una aplicación de escritorio y servidor para navegar páginas de [NomadNet](https://github.com/markqvist/NomadNet) a través de la red mesh [Reticulum](https://reticulum.network/).

2. [Instalación](installation.md): descargas, Docker y compilación desde el código fuente
3. [Configuración de Reticulum](reticulum-setup.md): archivos de configuración, interfaces y unirse a la mesh
4. [Uso del navegador](using-the-browser.md): pestañas, barra de direcciones, historial y favoritos
5. [Navegación y URLs](navigation-and-urls.md): destinos NomadNet, rutas y páginas integradas
6. [Discovery](discovery.md): encontrar nodos anunciados en tus interfaces

8. [Atajos de teclado](keyboard-shortcuts.md): combinaciones predeterminadas y cómo cambiarlas
9. [Extensiones](extensions.md): instalar plugins y escribir los tuyos
10. [Modo servidor](server-mode.md): hosting sin interfaz gráfica, Docker y proxies inversos
11. [Datos y perfiles](data-and-profiles.md): dónde viven tus datos y cómo funcionan los perfiles
12. [Seguridad](security.md): uso seguro, verificación de descargas e informes de problemas
13. [Solución de problemas](troubleshooting.md): problemas frecuentes y correcciones
