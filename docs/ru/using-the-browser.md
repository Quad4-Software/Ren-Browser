# Работа с браузером

Вкладки, адресная строка, история, избранное и просмотр страниц.

## Окно приложения

Основные области:

- **Панель вкладок** сверху. Перетаскивание, закрепление, разделённый вид.
- **Адресная строка** для назначений и встроенных схем.
- **Область контента** с отрендеренными страницами Micron.
- **Боковые панели**: Обнаружение, История, Настройки, DevTools и панели расширений.

На узких экранах часть элементов заменяет мобильная навигация.

## Вкладки

- **Новая вкладка**: `Ctrl+T` / `Cmd+T` (см. [Горячие клавиши](keyboard-shortcuts.md))
- **Закрыть вкладку**: `Ctrl+W` / `Cmd+W`
- **Восстановление сессии**: открытые вкладки сохраняются в SQLite и поднимаются при следующем запуске

Закреплённые вкладки остаются в начале списка. Разделённый вид показывает две вкладки рядом.

## Адресная строка

Введите назначение NomadNet или встроенную схему:

| Ввод | Результат |
|------|-----------|
| 32 hex-символа | Открывается `hash:/page/index.mu` |
| Полный URL NomadNet | Как введено |
| `about:` | О программе, версия и пути |
| `license:` | Текст лицензии |
| `editor:` | Редактор Micron |
| `settings` | Открывает Настройки |

Enter : переход. Фокус в строку: `Ctrl+L` / `Cmd+L`.

## Переход по ссылкам

Ссылки Micron на странице разрешаются относительно текущего назначения. Локальные `about:`, `license:` и `editor:` работают как обычная навигация.

Внешние mesh-ссылки используют синтаксис назначений Reticulum. При ошибке проверьте доступность узла.

## История

Панель **История** в боковой панели или по горячей клавише. Можно:

- искать по заголовку или URL
- видеть записи по датам
- открыть прошлую страницу в текущей или новой вкладке

История хранится локально в SQLite, кроме публичного серверного режима.

## Избранное

Сохраняйте частые узлы из контекстного меню страницы или Обнаружения. Избранное в той же базе, что история и вкладки.

## Поиск на странице

`Ctrl+F` / `Cmd+F` : поиск текста на текущей странице Micron. Совпадения подсвечиваются.

## Инструменты разработчика

`Ctrl+Shift+I` / `Cmd+Shift+I` : DevTools. Полезно для:

- времени рендера
- исходного текста страницы
- отладки панелей расширений

## Загрузки

Файлы из страниц или действий попадают в меню загрузок. На настольных ОС путь следует правилам системы.

## Ошибки страницы

При сбое загрузки показывается короткое сообщение. Частые причины:

- назначение недоступно
- неверный контент Micron
- Reticulum не подключён

После исправления связи нажмите **Обновить** (`Ctrl+R` / `Cmd+R`).

## Редактор Micron

`editor:` : встроенный редактор для черновиков перед публикацией на узле NomadNet.
