# Навигация и URL

Форматы адресов в адресной строке и правила нормализации.

## Назначения NomadNet

Полный URL выглядит так:

```
<32-hex-символа>:/page/path.mu
```

Пример:

```
a1b2c3d4e5f6789012345678abcdef01:/page/index.mu
```

Хеш : идентификатор назначения Reticulum в hex. Путь после `:/` : файл на узле NomadNet.

## Короткий хеш

Если ввести только 32 hex-символа, Ren Browser развернёт адрес в:

```
<hash>:/page/index.mu
```

Это стандартный путь к домашней странице NomadNet.

## Пути без хеша

При уже известном назначении путь с `/page/` может разрешаться относительно текущего узла. Для «холодного» перехода лучше полная форма `hash:/page/...`.

## Встроенные схемы

Обрабатываются внутри приложения, без mesh:

| Схема | Синонимы | Описание |
|-------|----------|----------|
| `about:` | `about` | Версия, сборка, пути к конфигу и данным |
| `license:` | `license` | Лицензия MIT |
| `editor:` | `editor` | Редактор исходников Micron |

Регистр не важен. Пробелы по краям обрезаются.

## Схемы расширений

Установленные плагины регистрируют свои схемы в `renbrowser.plugin.json`. Пример: `hello:` в hello-extension. См. [Расширения](extensions.md).

## Настройки и внутренние маршруты

Адресная строка ориентирована на mesh и встроенные схемы. **Настройки** : кнопка в боковой панели или `Ctrl+,` / `Cmd+,`.

## Заголовки вкладок

Источники заголовка:

1. метаданные страницы с узла
2. имя из Обнаружения для известного хеша
3. укороченный хеш или путь

## Записи истории

При загрузке контента создаётся запись с URL, заголовком, хешем и временем. Встроенные страницы вроде `about:` тоже попадают в историю.

## Ссылки на страницах Micron

При клике:

- `about:`, `license:`, `editor:` открываются локально
- абсолютные mesh-URL ведут напрямую
- относительные пути комбинируются с текущим назначением

## Правила нормализации

| Ввод | Нормализованный URL |
|------|---------------------|
| `about` | `about:` |
| `license` | `license:` |
| `editor` | `editor:` |
| `abcdef...` (32 hex) | `abcdef...:/page/index.mu` |
| Полный `hash:/page/foo.mu` | без изменений |
