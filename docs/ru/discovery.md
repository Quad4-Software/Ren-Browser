# Обнаружение

Обнаружение показывает узлы NomadNet, анонсированные в ваших интерфейсах Reticulum. Используйте, когда хеш назначения ещё неизвестен.

## Как открыть

- Кнопка **Обнаружение** в боковой панели
- Горячая клавиша (`Ctrl+Shift+D` / `Cmd+Shift+D` по умолчанию)

В списке: отображаемые имена, хеши и метаданные из анонса.

## Что попадает в список

Узел виден, когда:

1. Reticulum получил анонс для этого назначения
2. анонс распознан как NomadNet или совместимый тип
3. ваши интерфейсы достигают пути, по которому пришёл анонс

Список обновляется в реальном времени.

## Открыть узел

Клик по строке открывает страницу по умолчанию (обычно `index.mu`). Хеш можно скопировать в адресную строку.

## Избранное из Обнаружения

У многих строк есть действие добавить узел в избранное. Данные хранятся в локальном профиле.

## Пустой список

См. [Настройка Reticulum](reticulum-setup.md):

- интерфейсы должны быть активны в Настройках
- нужна связность с пирами, которые несут анонсы
- после подключения подождите немного

## Анонс и доступность

Узел в Обнаружении не гарантирует успешную загрузку каждой страницы. Для запросов страниц по LXMF через Reticulum всё равно нужен маршрут до назначения.

Если узел в списке, а страницы падают, проверьте транспорт и доступность удалённого узла.

## Сообщественные интерфейсы

В настройках могут быть общие описания интерфейсов. Они помогают подключиться к более широким сегментам mesh. Применяйте изменения в разделе Reticulum.
