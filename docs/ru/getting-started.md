# Начало работы

Ren Browser открывает страницы NomadNet через Reticulum. Это небольшой браузер для mesh, а не для обычного интернета.

## Что умеет Ren Browser

- Открывает страницы `.mu` и другой контент NomadNet с узлов mesh
- Показывает узлы, доступные через ваши интерфейсы Reticulum
- Хранит вкладки, историю, закладки и настройки на вашем компьютере (или в браузере в публичном серверном режиме)
- Рендерит разметку Micron через WASM-парсер и встроенные темы
- Поддерживает расширения: панели, URL-схемы и команды

## Что нужно заранее

Перед загрузкой удалённой страницы на той же машине (или в Docker для серверного режима) должен работать **Reticulum**.

По умолчанию Ren Browser читает конфиг Reticulum из `~/.reticulum-go/`. Другой файл задаётся через `--config` или переменную `REN_BROWSER_CONFIG`.

Для просмотра страниц NomadNet в mesh **не обязателен** доступ в интернет. Нужен хотя бы один интерфейс, через который видны другие узлы Reticulum.

## Настольная версия или сервер

| Режим | Когда использовать |
|------|---------------------|
| **Настольный** (по умолчанию) | Повседневная работа на Linux, Windows или macOS. Окно приложения, локальная база SQLite. |
| **Сервер** | Домашний сервер, Docker или машина, где уже крутится Reticulum. UI открывается в другом браузере по `http://host:8080`. |
| **Android** | Мобильные сборки, если в релизе есть APK. Те же функции в touch-интерфейсе. |

## Чеклист первого запуска

1. Установите Reticulum и положите конфиг в `~/.reticulum-go/`
2. Установите Ren Browser из [релизов](https://github.com/Quad4-Software/Ren-Browser/releases) или соберите из исходников
3. Запустите приложение и дождитесь подключения интерфейсов Reticulum
4. Откройте **Обнаружение** или введите 32-символьный хеш назначения в адресной строке
5. Откройте `about:` и проверьте версию, путь к конфигу и каталог данных

## Встроенные страницы (без mesh)

Работают даже без связи с mesh:

| Адрес | Назначение |
|-------|------------|
| `about:` | Версия, сборка, пути |
| `license:` | Текст лицензии MIT |
| `editor:` | Встроенный редактор Micron |

Введите `settings` в адресной строке или откройте настройки сочетанием клавиш.
