# Документация Ren Browser (русский)

Ren Browser : это настольное и серверное приложение для просмотра страниц [NomadNet](https://github.com/markqvist/NomadNet) через mesh-сеть [Reticulum](https://reticulum.network/).

3. [Настройка Reticulum](reticulum-setup.md): конфиг, интерфейсы и подключение к mesh
4. [Работа с браузером](using-the-browser.md): вкладки, адресная строка, история и избранное
5. [Навигация и URL](navigation-and-urls.md): адреса NomadNet, пути и встроенные страницы
6. [Обнаружение](discovery.md): поиск узлов в ваших интерфейсах Reticulum
7. [Настройки](settings.md): темы, интерфейсы, горячие клавиши и профили
8. [Горячие клавиши](keyboard-shortcuts.md): стандартные сочетания и их изменение
9. [Расширения](extensions.md): установка плагинов и создание своих
10. [Серверный режим](server-mode.md): headless, Docker и обратный прокси

14. [Разработка](development.md): локальная среда, задачи и проверки качества
15. [Архитектура](architecture.md): устройство приложения
16. [Участие в проекте](contributing.md): отправка патчей через Reticulum
17. [Частые вопросы](faq.md): краткие ответы
