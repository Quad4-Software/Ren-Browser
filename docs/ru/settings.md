# Настройки

Внешний вид, Reticulum и поведение браузера на вашей машине.

Откройте **Настройки** в боковой панели или `Ctrl+,` / `Cmd+,` (можно переназначить).

## Внешний вид

### Темы

Есть тёмная и светлая темы. Можно:

- переключить тему в Настройках
- импортировать JSON темы
- экспортировать текущую тему

Темы влияют на цвета интерфейса, шрифты и стиль страниц Micron где применимо.

### Оформление окна

На настольных системах: нативная заголовочная полоса или безрамочное окно с кнопками свернуть, развернуть и закрыть. В безрамочном режиме верхняя область перетаскивает окно.

## Reticulum

Раздел Reticulum показывает:

- активные интерфейсы и статус
- счётчики переданных и принятых байт
- редактор конфига с hot reload

После правок применяйте изменения из Настроек, когда это возможно, без полного перезапуска.

## Горячие клавиши

Для каждого действия задаётся аккорд, например `mod+t` для новой вкладки. `mod` : Ctrl на Windows и Linux, Cmd на macOS.

См. [Горячие клавиши](keyboard-shortcuts.md).

## Расширения

**Настройки → Расширения**:

- установка из zip, папки или собранного `.wasm`
- предпросмотр: разрешения, сетевые endpoint, значки подписи, предупреждения безопасности, языки UI
- включение и отключение
- статус подписи и предупреждения об изменении файлов

См. [Расширения](extensions.md).

## Профиль и данные

В настройках видны пути:

- база SQLite
- конфиг Reticulum
- каталог плагинов `~/.renbrowser/plugins/`

Именованные профили и импорт или экспорт: [Данные и профили](data-and-profiles.md).

## Параметры браузера

Дополнительные переключатели могут включать:

- нативный заголовок или безрамочный режим
- поведение панелей по умолчанию
- опции из сохранённых prefs в SQLite

Точный набор зависит от версии. Сверяйтесь с подписями в UI и этим документом для вашего тега релиза.

## Мобильная версия

На Android те же данные, интерфейс адаптирован под касания. Reticulum и темы доступны.
