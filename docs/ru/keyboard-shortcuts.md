# Горячие клавиши

Стандартные сочетания на настольных системах. Меняются в **Настройки → Горячие клавиши**.

`mod` : **Ctrl** на Windows и Linux, **Cmd** на macOS.

## Стандартные сочетания

| Действие | Аккорд | Описание |
|--------|-------|-------------|
| Фокус на адресную строку | `mod+l` | Курсор в поле URL |
| Обновить страницу | `mod+r` | Перезагрузка активной вкладки |
| Инструменты разработчика | `mod+shift+i` | DevTools |
| Поиск на странице | `mod+f` | Поиск по тексту |
| Панель Обнаружение | `mod+shift+d` | Боковая панель |
| Настройки | `mod+,` | Панель настроек |
| Новая вкладка | `mod+t` | Пустая вкладка |
| Новое окно | `mod+shift+n` | Второе окно (если поддерживается) |
| Закрыть вкладку | `mod+w` | Закрыть активную вкладку |
| Полный экран | `f11` | Переключить полноэкранный режим |

## Запись нового сочетания

1. **Настройки → Горячие клавиши**
2. Выберите действие
3. Нажмите новую комбинацию
4. Разрешите конфликты с другими действиями перед сохранением

Во время записи остальные сочетания временно отключены.

## Синтаксис аккорда

Строка в нижнем регистре, части через `+`:

- `mod` : Ctrl или Cmd
- `shift` : Shift
- `alt` : Alt
- последняя часть : имя клавиши (`l`, `r`, `,`, `f11` и т.д.)

Пример: `mod+shift+d` : Ctrl+Shift+D на Linux.

## Сочетания расширений

Плагины могут объявлять `keybind` в `renbrowser.plugin.json`. Включённые расширения добавляют команды в палитру и обработку клавиш.

## Android

При подключённой физической клавиатуре правила те же. Сенсорный UI не требует сочетаний.
