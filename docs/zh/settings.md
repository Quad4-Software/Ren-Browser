# 设置

设置控制 Ren Browser 的外观、与 Reticulum 的连接方式以及在你的机器上的行为。

通过侧边栏或使用 `Ctrl+,` / `Cmd+,`（可在快捷键设置中自定义）打开设置。

## 外观

### 主题

Ren Browser 内置深色和浅色主题。你可以：

- 在设置中切换主题
- 导入主题 JSON 文件
- 导出当前主题用于备份或共享

主题影响界面颜色、排版标记以及适用的 Micron 页面样式。

### 窗口外框

在桌面模式下，你可以选择原生标题栏或带自定义最小化、最大化和关闭控件的无边框窗口。无边框模式在窗口顶部使用可拖动区域。

## Reticulum

Reticulum 部分显示：

- 活动接口及其状态
- 发送和接收字节计数器
- 支持热重载的配置编辑器

编辑配置后，尽量从设置中应用变更，而不是重启应用。

## 快捷键绑定

每个操作可以有一个组合键，例如 `mod+t` 表示新建标签页。`mod` 在 Windows 和 Linux 上表示 Ctrl，在 macOS 上表示 Cmd。

默认绑定及如何录制新绑定请参阅[键盘快捷键](keyboard-shortcuts.md)。

## 扩展

通过**设置 → 扩展**管理插件：

- 从 zip 压缩包、文件夹或捆绑的 `.wasm` 模块安装
- 查看安装预览：权限、网络端点、签名徽章、安全说明、捆绑的 UI 语言
- 启用或禁用已安装的扩展
- 查看每张卡片上的签名状态和篡改警告

插件清单格式、签名及安装路径请参阅[扩展](extensions.md)。

## 配置文件与数据

设置中显示以下路径：

- SQLite 数据库位置
- Reticulum 配置路径
- `~/.renbrowser/plugins/` 下的插件目录

命名配置文件及导入导出请参阅[数据与配置文件](data-and-profiles.md)。

## 浏览器偏好

其他选项可能包括：

- 原生标题栏与无边框窗口
- 默认面板行为
- 与 SQLite 中存储的浏览器偏好同步的选项

具体选项可能因版本而异。如有疑问，请查看 UI 中的标签以及对应版本标签的此文档。

## 移动端布局

在 Android 上，设置使用相同的数据，但可能会为触控导航重新分组项目。核心 Reticulum 和主题选项仍然可用。
