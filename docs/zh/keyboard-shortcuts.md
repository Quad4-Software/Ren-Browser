# 键盘快捷键

桌面端的默认快捷键绑定。你可以在**设置 → 快捷键绑定**中修改。

`mod` 在 Windows 和 Linux 上表示 **Ctrl**，在 macOS 上表示 **Cmd**。

## 默认绑定

| 操作 | 组合键 | 描述 |
|------|--------|------|
| 聚焦地址栏 | `mod+l` | 将键盘焦点移至 URL 输入框 |
| 重新加载页面 | `mod+r` | 重新加载当前标签页 |
| 开发工具 | `mod+shift+i` | 打开或关闭开发工具 |
| 页面内查找 | `mod+f` | 在当前页面打开查找栏 |
| 发现面板 | `mod+shift+d` | 打开发现侧边栏 |
| 设置面板 | `mod+,` | 打开设置 |
| 新建标签页 | `mod+t` | 打开空白标签页 |
| 新建窗口 | `mod+shift+n` | 打开另一个窗口（支持时） |
| 关闭标签页 | `mod+w` | 关闭当前标签页 |
| 全屏 | `f11` | 切换全屏模式 |

## 录制新绑定

1. 打开**设置 → 快捷键绑定**
2. 点击你想更改的操作
3. 按下新的组合键
4. UI 中会显示与其他操作的冲突。保存前请先解决冲突。

录制期间，其他快捷键会暂停，使录制器只捕获你的新组合键。

## 组合键语法

绑定以小写组合键形式存储，各部分之间用 `+` 连接：

- `mod`：Ctrl 或 Cmd
- `shift`：Shift
- `alt`：Alt
- 最后一段是键名（`l`、`r`、`,`、`f11` 等）

示例：`mod+shift+d` 在 Linux 上为 Ctrl+Shift+D。

## 扩展快捷键绑定

扩展可以通过 `renbrowser.plugin.json` 中的可选 `keybind` 字段为命令贡献快捷键。启用的扩展会将其命令合并到主机命令面板和快捷键处理中。

## Android

连接硬件键盘的 Android 设备遵循相同的组合键规则。触控 UI 不需要快捷键。
