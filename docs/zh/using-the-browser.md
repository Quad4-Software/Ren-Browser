# 使用浏览器

本页介绍日常使用：标签页、地址栏、历史记录、收藏夹及页面查看。

## 窗口布局

主要区域包括：

- 顶部的**标签栏**。可拖动标签页、固定标签页或打开分屏视图。
- 用于目标地址和内置方案的**地址栏**。
- 渲染 Micron 页面的**内容区域**。
- 用于发现、历史记录、设置、开发工具和扩展面板的**侧边面板**。

在较小的屏幕上，移动端导航栏会替代部分桌面界面元素。

## 标签页

- **新建标签页**：默认快捷键 `Ctrl+T` / `Cmd+T`（参见[键盘快捷键](keyboard-shortcuts.md)）
- **关闭标签页**：`Ctrl+W` / `Cmd+W`
- **恢复会话**：打开的标签页保存在本地数据库中，下次启动时自动恢复

固定的标签页位于标签栏前端。分屏视图允许并排显示两个标签页。

## 地址栏

输入 NomadNet 目标地址或使用内置方案：

| 输入 | 结果 |
|------|------|
| 32 字符十六进制哈希值 | 打开 `hash:/page/index.mu` |
| 完整的 NomadNet URL | 按原样打开 |
| `about:` | 包含版本和路径的关于页面 |
| `license:` | 许可证文本 |
| `editor:` | Micron 编辑器 |
| `settings` | 打开设置（通过 UI 路由） |

按 Enter 键导航。也可以通过 `Ctrl+L` / `Cmd+L` 聚焦地址栏。

## 跟随链接

页面上的 Micron 链接相对于当前目标地址解析。内部的 `about:`、`license:` 和 `editor:` 链接与普通导航一样工作。

外部网状网络链接使用 Reticulum 目标地址语法。如果链接失败，检查目标节点是否可达。

## 历史记录

从侧边栏或通过快捷键打开历史记录面板。你可以：

- 按标题或 URL 搜索
- 查看按日期分组的条目
- 在当前标签页或新标签页中打开历史页面

历史记录存储在本地 SQLite 中（桌面模式），除非你使用服务器公共模式。

## 收藏夹

通过页面右键菜单或发现面板将常访问的节点保存为收藏夹。收藏夹与历史记录和标签页使用相同的存储同步。

## 页面内查找

按 `Ctrl+F` / `Cmd+F` 在当前 Micron 页面上搜索文本。匹配项会在内容查看器中高亮显示。

## 开发工具

按 `Ctrl+Shift+I` / `Cmd+Shift+I` 打开开发工具。适用于：

- 检查渲染时间
- 查看原始页面源码
- 调试提供开发工具条目的扩展面板

## 下载

当页面或操作提供文件时，项目会显示在下载菜单中。桌面模式下路径遵循你的操作系统下载规范。

## 页面错误

如果页面无法加载，你会看到带有简短消息的错误状态。常见原因：

- 目标地址不可达
- 无效的 Micron 内容
- Reticulum 未连接

修复连接问题后，使用**重新加载**（`Ctrl+R` / `Cmd+R`）。

## Micron 编辑器

打开 `editor:` 在内置编辑器中编写 Micron。在发布到 NomadNet 节点之前，可用于本地草稿。
