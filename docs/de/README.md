# Ren Browser Dokumentation (Deutsch)

Ren Browser ist eine Desktop- und Server-Anwendung zum Durchsuchen von [NomadNet](https://github.com/markqvist/NomadNet)-Seiten über das [Reticulum](https://reticulum.network/)-Mesh-Netzwerk.

11. [Daten und Profile](data-and-profiles.md): wo Ihre Daten liegen und wie Profile funktionieren
