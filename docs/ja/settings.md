# 設定

設定では、Ren Browser の外観、Reticulum への接続方法、マシン上での動作を制御します。

サイドバーから、または `Ctrl+,` / `Cmd+,`（キーバインド設定でカスタマイズ可能）で設定を開きます。

## 外観

### テーマ

Ren Browser にはダークテーマとライトテーマが付属しています。以下のことができます：

- 設定でテーマを切り替える
- テーマ JSON ファイルをインポートする
- バックアップや共有のために現在のテーマをエクスポートする

テーマはクロームの色、タイポグラフィトークン、該当する場合は Micron ページのスタイリングに影響します。

### ウィンドウクロム

デスクトップでは、ネイティブタイトルバーと、カスタムの最小化・最大化・閉じるコントロールを持つフレームレスウィンドウを選択できます。フレームレスモードではウィンドウ上部にドラッグ可能な領域を使用します。

## Reticulum

Reticulum セクションには以下が表示されます：

- アクティブなインターフェースとその状態
- 送受信バイトカウンター
- ホットリロード付きの設定エディタ

設定を編集した後は、可能な限りアプリを再起動するのではなく、設定から変更を適用してください。

## キーバインド

各アクションには `mod+t`（新しいタブ）のようなコードを設定できます。`mod` は Windows および Linux では Ctrl、macOS では Cmd を意味します。

デフォルトと新しいバインドの記録方法については [キーボードショートカット](keyboard-shortcuts.md) を参照してください。

## 拡張機能

**設定 → 拡張機能** からプラグインを管理します：

- zip アーカイブ、フォルダー、またはバンドルされた `.wasm` モジュールからインストールする
- インストールプレビューを確認する：権限、ネットワークエンドポイント、署名バッジ、セキュリティ注意事項、バンドルされた UI 言語
- インストール済みの拡張機能を有効化または無効化する
- 各カードの署名ステータスと改ざん警告を表示する

マニフェスト形式、署名、インストールパスについては [拡張機能](extensions.md) を参照してください。

## プロファイルとデータ

設定には以下のパスが表示されます：

- SQLite データベースの場所
- Reticulum 設定のパス
- `~/.renbrowser/plugins/` 以下のプラグインディレクトリ

名前付きプロファイルとインポート・エクスポートについては [データとプロファイル](data-and-profiles.md) を参照してください。

## ブラウザの設定

追加のトグルには以下が含まれることがあります：

- ネイティブタイトルバーとフレームレスウィンドウ
- デフォルトのパネル動作
- SQLite に保存されたブラウザの設定と同期するオプション

正確なトグルはバージョンによって異なる場合があります。不明な場合は、UI のラベルとリリースタグ用のこのドキュメントを確認してください。

## モバイルレイアウト

Android では、設定は同じデータを使用しますが、タッチナビゲーション用にアイテムがグループ化されることがあります。Reticulum のコアオプションとテーマオプションは引き続き利用できます。
