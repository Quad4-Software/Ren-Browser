# はじめに

Ren Browser を使うと、Reticulum 上の NomadNet ページを開くことができます。公共のウェブではなく、メッシュネットワーク向けに作られた小さなブラウザと考えてください。

## Ren Browser でできること

- メッシュのデスティネーションにある `.mu` ページやその他の NomadNet コンテンツを開く
- Reticulum インターフェース経由で到達可能なノードを表示する
- タブ、履歴、ブックマーク、設定をマシン上に保存する（公開サーバーモード利用時はブラウザに保存）
- WASM パーサーと組み込みテーマで Micron マークアップをレンダリングする
- パネル、URL スキーム、コマンドを追加する拡張機能をサポートする

## 事前に必要なもの

Ren Browser がリモートページを読み込むには、同じマシン上（またはサーバーモード用の Docker コンテナにマウントされた環境）で動作する **Reticulum** のセットアップが必要です。

Ren Browser はデフォルトで `~/.reticulum-go/` から Reticulum の設定を読み込みます。`--config` フラグまたは `REN_BROWSER_CONFIG` 環境変数で別のファイルを指定することもできます。

メッシュ上の NomadNet ページを閲覧するために従来のインターネット接続は**不要**です。ただし、他の Reticulum ノードに到達できるインターフェースが少なくとも 1 つ必要です。

## デスクトップとサーバー

| モード | 最適な用途 |
|--------|-----------|
| **デスクトップ**（デフォルト） | Linux、Windows、macOS での日常利用。ネイティブウィンドウ、ローカル SQLite データベース。 |
| **サーバー** | ホームラボ、Docker、または既に Reticulum が動作しているマシン。`http://host:8080` で別のブラウザから Ren Browser を開く。 |
| **Android** | リリースに APK が含まれている場合のモバイルビルド。タッチレイアウトで同じコアブラウザ機能を提供。 |

## 初回起動チェックリスト

1. Reticulum をインストールし、`~/.reticulum-go/` に設定を作成またはコピーする
2. [リリース](https://github.com/Quad4-Software/Ren-Browser/releases)から Ren Browser をインストールするか、ソースからビルドする
3. Ren Browser を起動し、インターフェース上で Reticulum が接続するのを待つ
4. **ディスカバリー** を開くか、アドレスバーに 32 文字のデスティネーションハッシュを入力する
5. `about:` にアクセスして、バージョン、設定パス、データディレクトリを確認する

## 組み込みページ（メッシュ不要）

メッシュからオフラインの場合でも動作します：

| アドレス | 用途 |
|---------|------|
| `about:` | アプリのバージョン、ビルド情報、パス |
| `license:` | MIT ライセンステキスト |
| `editor:` | 組み込み Micron エディタ |

アドレスバーに `settings` と入力するか、設定のショートカットを押して設定を開きます。
