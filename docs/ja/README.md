# Ren Browser ドキュメント（日本語）

Ren Browser は、[Reticulum](https://reticulum.network/) メッシュネットワーク上で [NomadNet](https://github.com/markqvist/NomadNet) のページを閲覧するためのデスクトップ・サーバーアプリケーションです。

3. [Reticulum の設定](reticulum-setup.md)：設定ファイル、インターフェース、メッシュへの参加
4. [ブラウザの使い方](using-the-browser.md)：タブ、アドレスバー、履歴、お気に入り

7. [設定](settings.md)：テーマ、インターフェース、キーバインド、プロファイル

13. [トラブルシューティング](troubleshooting.md)：よくある問題と解決策
14. [開発](development.md)：ローカル開発環境、タスク、品質チェック
