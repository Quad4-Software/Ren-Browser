# Ren Browser documentation (English)

Ren Browser is a desktop and server app for browsing [NomadNet](https://github.com/markqvist/NomadNet) pages over the [Reticulum](https://reticulum.network/) mesh network.

4. [Using the browser](using-the-browser.md): tabs, address bar, history, and favorites
5. [Navigation and URLs](navigation-and-urls.md): NomadNet destinations, paths, and built-in pages
6. [Discovery](discovery.md): finding nodes announced on your interfaces
7. [Settings](settings.md): themes, interfaces, keybinds, and profiles
8. [Keyboard shortcuts](keyboard-shortcuts.md): default bindings and how to change them

10. [Server mode](server-mode.md): headless hosting, Docker, and reverse proxies
11. [Data and profiles](data-and-profiles.md): where your data lives and how profiles work
12. [Security](security.md): safe use, verifying downloads, and reporting issues
13. [Troubleshooting](troubleshooting.md): common problems and fixes
14. [Development](development.md): local dev setup, tasks, and quality checks
15. [Architecture](architecture.md): how the app is put together
16. [Contributing](contributing.md): sending patches over Reticulum
