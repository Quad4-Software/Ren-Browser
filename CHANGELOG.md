# Changelog

All notable changes to Ren Browser are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

Planned first release: **0.1.0**

### Added

- Wails v3 desktop shell with Svelte 5 frontend
- NomadNet page browsing over Reticulum (`quad4/reticulum-go`)
- Micron-first rendering via `micron-parser-go`
- Tabbed browser chrome with discovery, history, devtools, and settings
- SQLite persistence for nodes, history, favorites, and tabs
- Dark/light themes with JSON import/export
- Headless server mode and Docker deployment path
- Linux AppImage, Windows, and macOS packaging tasks
- `task check`, interop tests, gosec, and frontend quality gates
- GitHub Actions CI with Go, frontend, security, server, and Linux desktop checks
- Nightly, beta, and stable release workflows with multi-platform artifacts
- GHCR publishing for the server image
- History panel search and date separators
- Custom frameless window controls (minimize, maximize, close)
- Settings toggle switches
- Per-interface Reticulum hot reload with tx/rx stats in Settings
